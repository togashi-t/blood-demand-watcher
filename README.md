# blood-demand-watcher

## 概要
日本赤十字社の全国7ブロックのウェブサイトから、現在各型の血液をどれだけ必要としているかの情報をスクレイピングして出力するツールです。

## 依存関係
- Scala 3.3.7
- [sttp client](https://sttp.softwaremill.com/) (v4) - HTTPリクエスト用
- [jsoup](https://jsoup.org/) - HTMLパース用

## 使い方
### 実行方法
sbtがインストールされている環境で、以下のコマンドを実行してください：

```bash
sbt run
```

### 出力例
実行すると、以下のような形式で各地域の状況が表示されます：

```
=== 北海道ブロック血液センター｜日本赤十字社 ===
URL: https://www.bs.jrc.or.jp/hkd/bbc/
  A : 非常に困ってます
  O : 非常に困ってます
  B : 非常に困ってます
  AB: 安心です

=== 東北ブロック血液センター｜日本赤十字社 ===
URL: https://www.bs.jrc.or.jp/th/bbc/
  A : 非常に困ってます
  O : 非常に困ってます
  B : 困っています
  AB: 心配です

=== 関東甲信越ブロック血液センター｜日本赤十字社 ===
URL: https://www.bs.jrc.or.jp/ktks/bbc/
...
```

## 注意事項
- 本ツールはウェブスクレイピングを行っています。実行時には対象サイトへの負荷に配慮し、リクエスト間に待機時間を設けています。
- 対象サイトの構造が変更された場合、正しく取得できなくなる可能性があります。
